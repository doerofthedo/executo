# Executo

Initial monorepo scaffold for the Executo bailiff district debtor manager platform.

## Structure

- `backend/` Laravel-oriented API application scaffold
- `frontend/` Vue 3 + Vite SPA scaffold
- `public/` Apache document root
- `docker/` Local development environment

## Next Steps

1. Install backend dependencies in `backend/` with Composer.
2. Install frontend dependencies in `frontend/` with npm.
3. Start the Docker environment from `docker/`.
4. Implement the domain actions, policies, form requests, and API resources.
